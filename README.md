## HOW TO START (video for fast)

`.bat` for fast

1.  **Python 3.10+** required.
2.  Install requirements: `pip install -r requirements.txt`
3.  Click `run.bat`.
4.  Video: [tutorial](https://youtu.be/5tMwjg5hXNY)

## ‚öôÔ∏è SETTINGS (STOP EDITING JSON FILES)

You don't need to touch `Config.json` like a caveman anymore. 

Inside the app, there is a **[SETTINGS]** button. Click it.
You can change everything there. But if you are confused, here is what the buttons do:

| Setting | Translation for Dummies |
| :--- | :--- |
| **Month Names** | `true` = "January", `false` = Numbers. Aesthetic choice. |
| **No Year Folders** | If `true`, it dumps everything in one place. Chaotic evil. |
| **Skip Highlights Covers** | Saves space. Who looks at covers anyway? |
| **Flatten Highlights** | `true` = Removes folders for highlights. Total mess. Don't do it. |
| **Disable Text Files** | `true` = Creates specific folders (Photos/Videos). `false` = Dumps everything in a "Post" folder with the caption text. I keep this ON (true) cause I'm organized. |
| **Download Tagged** | Downloads SPAM/ADS (#ad, #adv). Keep it `false` unless you love commercials. |
| **Merge Tagged** | Mixes photos/videos from ads in one folder. |
| **Download Labels** | Downloads custom lists/labels from the creator. |
| **Workers (Threads)** | Speed. Default is 5. If you have a potato PC, don't touch it. |
